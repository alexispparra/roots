# ROOTS App

Roots App

Esta es una aplicación React para gestión de proyectos, diseñada con una paleta de colores Earthy Sage Green.

## ¿Cómo ejecutar la app?

1. **Instala dependencias** (si no lo hiciste):
   ```bash
   npm install
   ```

2. **Inicia la app**:
   ```bash
   npm start
   ```
   Esto abrirá la app en tu navegador en `http://localhost:3000`.

## Notas importantes
- **NO ejecutes archivos individuales con Node.js** (por ejemplo, `node src/appTheme.js`). Solo usa `npm start`.
- En el entorno de preview de Windsurf, los datos pueden no persistir tras recargar la página.

## Estructura del código
- `src/appTheme.js`: Paleta de colores global.
- `src/App.js`: Componente principal y ruteo.
- `src/components/Sidebar.js`: Barra lateral de navegación.
- `src/components/ProjectList.js`: Listado y gestión de proyectos.

## ¿Problemas?
Si tienes algún error, copia el mensaje exacto y consulta aquí o revisa la configuración de tu entorno Node/React.
